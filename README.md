## 농축산물 직거래 사이트(ALT)

### 소개
ALT는 기존 쇼핑몰들과는 달리 판매자와 소비자간 채팅을 이용해 가격, 수량을 조정할 수 있습니다.




## 기술 스택

- Front-End
  - HTML, CSS, JavaScript
  - JQuery
  - BootStrap
- Back-End
  - Java
  - Mybatis
- WebServer
  - Apache-Tomcat 9.0
- DataBase
  - Oracle 11g
- Framework
  - Spring
